# HealthAI — Production Grade AI Health Backend

Full stack AI-powered health and fitness platform.

## Tech Stack
- Java Spring Boot 3.x
- Python FastAPI
- Gemini AI + LangChain + RAG
- Redis + Kafka + MySQL
- Docker Compose

## Services
- Auth Service — JWT + Refresh Token + Redis
- User Service — Profile + Health + BMI Calculator
- Workout Service — Logging + Stats + Streak
- AI Service — Gemini + ChromaDB + RAG

## Run
```bash
docker-compose up --build
```
